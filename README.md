# Results Summary Component

A responsive and accessible Results Summary Component built as part of the Frontend Mentor and Bristol SU - Web & Digital Content Coordinator role challenge. This project focuses on semantic HTML, responsive design, accessibility, and dynamic content rendering using JavaScript and JSON data.

## Features

* Responsive mobile-first design
* Dynamic rendering using local JSON data
* Semantic HTML structure
* Accessible keyboard navigation and focus states
* Interactive hover states
* Custom CSS Properties
* Clean and maintainable code

## Built With

* HTML5
* CSS3
* JavaScript (ES6+)
* Mobile-First Workflow

## Key Learnings

* Building responsive layouts across multiple screen sizes
* Rendering UI dynamically from JSON data
* Creating scalable CSS using design tokens
* Implementing accessibility best practices
* Structuring maintainable frontend code
